## 实验题⽬
基于随机滑动 FrozenLake 的 Q-learning 求解实验

## 实验⽬的
- 理解 Gymnasium 中 FrozenLake 环境的状态、动作、奖励与终⽌机制。
- 掌握表格型 Q-learning 的基本实现⽅法。
- 观察在随机滑动条件下，探索机制对学习效果的影响。
- ⽐较在 88 预设地图 与 ⾃定义 1212 地图 上，Q-learning 的训练难度与最终表现差异。
FrozenLake-v 1 是 Gymnasium Toy Text 中的经典离散环境，⽀持预设 4x4 、8x8 地图，也⽀持通过 is_slippery=True 时，动作具有随机性，不⼀定严格朝期望⽅向移动。
使⽤ Gymnasium 预设地图：env = gym.make ("FrozenLake-v 1", map_name="8 x 8", is_slippery=True)

## 实验要求 ：

### 1. 实现 Q-learning
需要⾃⾏实现以下部分：
- Q 表初始化
- epsilon-greedy 动作选择
- Q-learning 更新公式
- 训练过程
- 测试过程
核⼼更新公式为：Q(s,a) ← Q(s,a) + α [R + γ max_a' Q(s',a') - Q(s,a)]
FrozenLake 的 Gymnasium 官⽅教程就是使⽤表格型Q-learning 来求解该环境。也可以进⾏查阅

### 2. 必须启⽤随机滑动
两种环境都必须设置：
```python
is_slippery=True
```
官⽅⽂档说明，在 is_slippery=True 时，智能体会以⼀定概率滑向⾮预期⽅向；例如朝某个⽅向⾏动时，不⼀定真的按该⽅向前进。⽂档给出的说明是：在滑动模式下，智能体会以 1/3 的概率朝期望⽅向移动，并以 1/3 的概率朝两个垂直⽅向之⼀移动

### 3. ⽐较两种地图上的学习效果
⾄少⽐较以下指标：
- 平均回报
- 成功率
- 平均步数
- 训练曲线变化趋势

### 4. 输出最终策略
训练结束后，需要输出每个状态的贪婪策略，例如使⽤箭头表⽰：
- 0：左
- 1：下
- 2：右
- 3：上
FrozenLake 的动作空间是离散 4 动作。

### 5. 可视化结果
⾄少完成以下⼀种可视化：
- 训练奖励曲线
- 成功率曲线
- 最终策略⽹格图
- 动态效果图
## 建议参数范围
这些是你可以在实验要求⾥给出的建议值，不是唯⼀答案：
- 学习率 alpha : 0.05 ～ 0.2
- 折扣因子 gamma : 0.95 ～ 0.99
- 初始探索率 epsilon : 1.0
- 最小探索率 epsilon_min : 0.01 ～ 0.05
- 探索衰减 epsilon_decay : 0.995 ～ 0.9995
- 训练回合数：8x8：至少 5000 回合
- 12x12：至少 10000 回合
由于 is_slippery=True 且地图规模较⼤，本实验训练难度明显⾼于 4x4 的基础版本，因此训练轮数应适当增加
